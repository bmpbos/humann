Misc data files for working with HUMAnN2 tools

For use with rename_table.py:
map_uniref50_name.txt.gz: human-readable names for UniRef50 centroids
map_ko_name.txt.gz: human-readable names for KEGG Orthogroups (legacy function)

For use with regroup_table.py:
map_uniref50_ko.txt.gz: map UniRef50 centroids to KEGG Orthogroups
map_infogo1000_uniref50.txt.gz: map informative Gene Ontology subset to UniRef50 orthogroups (use --reverse flag)
map_transporter_uniref50.txt.gz: map MetaCyc transporters to UniRef50 orthogroups (use --reverse flag)
